Readme file for catalog analysis using obspy and pandas
#####################################################
The files in this folder contains the tutorial of 
using Obspy and pandas to do some basic catalog 
analysis

Author: Qingkai Kong, qingkai.kong@gmail.com

Modify history:
(1) Created by 2014-04-19

#####################################################
Goal this time:

(1) Find the catalog of the earthquakes
(2) Do some fun stuff



#####################################################
Structure of the folder

World_Seismicity.ipynb - this is the ipython notebook
    contains the script to do a world seismicity.

la_habra_eq.ipynb - this is the ipython notebook 
    shows an example of doing basic catalog analysis
    using the La Habra earthquake

cat_analysis.py - script contains all the basic 
    functions to do the basic analysis.
    
    Note: This tutorial need pandas as a dependent.
