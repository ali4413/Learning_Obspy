Readme file for plotting the One Month aftershocks on the nice plot. 
#####################################################
The files in this folder contains all the files needed
to plot the one month seismicity after the Loma Prieta
earthquake

Author: Qingkai Kong, qingkai.kong@gmail.com
Modify history:
(1) Created by 2014-10-12

#####################################################
Purpose of this script to show you how to plot nice
maps for papers using Basemap

#####################################################
Structure of the folder

Plot_nice_maps.ipynb - this is the ipython notebook
    contains the main script to plot the figure, run 
    thecells in it step by step

LomaPrieta_OneMonth_M2_above.txt - this file contains 
    the ANSS catalog file you searched online, just
    copy it here, you don't need to modify anything. 
    
calif-fault.gmtlin - this contains the x, y, z of the
    CA faults
    
srtm_12_05.tif.grd - this is the GMT grid file converted
    from geotiff from srtm

example.png - this is the example figure generated by 
    this script

#####################################################
Dependencies:
netCDF4

just 'sudo pip install netCDF4'

#####################################################
Note: In order to get the nice Topographic, you need download the geotiff from 
http://srtm.csi.cgiar.org/SELECTION/inputCoord.asp

I converted the geotiff to GMT grid files.
